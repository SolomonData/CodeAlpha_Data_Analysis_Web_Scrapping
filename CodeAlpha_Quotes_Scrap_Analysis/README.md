# CODEALPHA Data Analysis Internship Task(Web Scrapping)

## Project Overview

This project focuses on scrapping dataset from the web, extracting it into a csv file, cleaning the extracted csv to perform Exploratory Data Analysis(EDA) and visual the Data using a comprehensive dashboard.

This project was completed as part of my **CODEALPHA Data Analytics Internship**.

## Dataset Used

- Quotes (from (https://quotes.toscrape.com/))

## Tools Used

- Python
- BeautifulSoup
- Pandas
- Matplotlib
- Plotly
- Jupyter Notebook
- Power Bi

## Problem Solved

This project answers the following questins:

- Which Author contributes the quotes?
- Which Quote tag is often used?
- Is Quotes evenly distributed among authors?
- Are the quotes suitable for quick readings and social media posts?

## Attached Files

- Top 100 Quotes to scrap (csv).
- Author Summary table.
- Web scrapping Notbook (ipynb).
- Power Bi Dashboard and Insights.
- Power Bi file.
